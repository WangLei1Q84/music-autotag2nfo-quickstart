# 增强能力说明

增强版不是“另一套软件”，而是在现有控制台基础上，把几项更重度的整理能力打开。

如果你已经跑通免费版，但开始觉得“基本够用，还差一点”，通常就是该看增强版的时候。

## 免费版 vs 增强版

| 能力 | 免费版 | 增强版 |
| --- | --- | --- |
| 增强识别 | 不可用 | 可用 |
| 手工元数据维护 | 不可用 | 可用 |
| 批量增强任务 | 不可用 | 可用 |
| 在线歌词补全 | 不可用 | 可用 |
| 在线封面补全 | 不可用 | 可用 |

## 这 5 项分别解决什么问题

### 1. 增强识别

适合“基础识别能跑，但总还有一批结果不够稳”的场景。

### 2. 手工元数据维护

适合“自动结果已经接近正确，但还想自己修一下细节”的场景。

### 3. 批量增强任务

适合“不只是改一两首，而是想集中处理一批内容”的场景。

### 4. 在线歌词补全

适合“本地源里没有歌词，但你希望媒体库尽量完整”的场景。

### 5. 在线封面补全

适合“希望媒体库看起来更完整，不想一堆纯文件名”的场景。

## 什么情况下值得升级

你已经跑通免费版，并出现以下任一情况时，就值得看增强版：
- 自动结果大体够用，但还需要人工精修
- 识别失败样本开始积累
- 想批量提升已有结果质量
- 开始在意歌词和封面完整度
