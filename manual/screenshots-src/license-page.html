授权管理

先确认实例绑定参数，再粘贴 license 并点击“保存并校验授权”。
示意图，仅用于手册说明
实例 IP
203.0.113.10
实例标识
https://demo.example.com
离线 license
MUSIC-AUTOTAG2NFO-LICENSE PACKAGE=enhanced INSTANCE_IP=203.0.113.10 INSTANCE_MARK=https://demo.example.com EXPIRES_AT=2027-12-31 SIGNATURE=...
授权状态：已生效
当前套餐：增强版
已开放能力数量：5
如果这里没有生效，先检查三件事：
1. 实例 IP 是否和 license 一致。
2. 实例标识是否和 license 一致。
3. 当前镜像版本是否已经包含最新授权逻辑。
