# 增强版能力说明

增强版不是“神秘高级模式”。

当前已经落地、可以明确展示的增强能力有 5 项，而且它们都已经在主项目控制台里有对应的能力边界。

## 免费版 vs 增强版

| 能力 | 免费版 | 增强版 |
| --- | --- | --- |
| 增强识别 | 不可用 | 可用 |
| 手工元数据维护 | 不可用 | 可用 |
| 批量增强任务 | 不可用 | 可用 |
| 在线歌词补全 | 不可用 | 可用 |
| 在线封面补全 | 不可用 | 可用 |

## 这 5 项分别解决什么问题

### 1. 增强识别

适合“基础识别不够稳，还需要更强补充链路”的场景。

### 2. 手工元数据维护

适合“自动结果已经接近正确，但需要人工修一把”的场景。

### 3. 批量增强任务

适合“不是只修一条，而是一次性处理一批轨迹”的场景。

### 4. 在线歌词补全

适合“本地源里没有歌词，但你希望尽量把歌词补完整”的场景。

### 5. 在线封面补全

适合“希望媒体库展示更完整，减少纯文件状态”的场景。

## 什么时候建议升级

你已经跑通免费版，并出现以下任一情况时，就值得看增强版：

- 自动结果够用，但还需要人工精修
- 识别失败样本开始积累
- 想批量提升已有结果质量
- 开始在意歌词和封面完整度
